Basic Usage

Depending on what type of device you're using to control the HQS, when you connect to the HQS through your device's web browser you should see a screen that looks something like this.

Read on to learn how to use the HQS.

Step 1 - Align The Laser

Place the HQS over the fastener hole to be inspected, and in your control device's web browser press the Align Laser button.

After the laser activates, move it to the position you'd like to start the scan. When the laser marks the start location, press the Stop button on your screen to finish.

Step 2 - Choose Output

By default HQS returns the complete data file with a scan, but you can also choose to have a sample color plot of the results returned as a picture. Check one or both of Generate Image to get a plot and Get Raw Data to get the data file.

Some versions of HQS will also have a Scan Comments text input, where you can optionally insert a comment that will be added to the output data file's header, e.g. part/structure being inspected, inspector's name, and so on. If your HQS has this feature, you can also leave it blank and the HQS will default to inserting the current date and time in the data file.

The plot generated by the HQS takes each of the height readings at each point in your fastener scan, and plots the point with a color proportional to the height reading. The plot is a simple PNG bitmap that you can view in your browser. Here's a sample plot.

Heads up - generating the plot can take a fair amount of time if your scan is large. TRI recommends retrieving the data file alone, and generating the plot offline.

Step 3 - Start Scanning

To begin acquiring fastener hole data, press the Start Scan button.

Depending on how your HQS has been configured, you may or may not see the laser line - if your HQS has been configured to trigger on movement, you'll only see the laser when you move the HQS. Complete the scan, and press the Stop button when you're finished.

Step 4 - Get Results

After you press the Stop button to finish a scan, the HQS will begin to process your scan. When finished, the web page will update with links to the type(s) of output you requested, along with a message indicating whether the scan was successfully completed or not.

To download the data file and/or the generated plot, just open the link(s) as you would with any other web browser. You should download a copy of any data you need to keep, since HQS only provides temporary storage for scans and plot images.
